## 10. Expansion Rule for Future Tools

Every future tool should get this tree:

```txt
Tool Page
|
+-- 3-5 how-to guides
+-- 2-4 problem pages
+-- 2-3 comparison pages
+-- 3-5 use-case pages
+-- 2-4 platform/device tutorials if intent exists
+-- 1 FAQ hub
+-- 1-3 glossary/explainer pages
```

Do not create all branches immediately. Create pages only when Search Console, competitor gaps or keyword research prove there is demand.
