# LiftPDF SEO Monitoring

Date: 2026-07-10

Status: weekly operating process.

## Purpose

LiftPDF growth must now be driven by real data, not guesses.

The weekly SEO process answers four questions:

1. Are pages indexed?
2. Which queries are gaining impressions?
3. Which pages have weak CTR?
4. Which pages deserve content, title, FAQ or internal-link updates?

## Weekly Search Console Checklist

Review every Monday:

| KPI | Where | What to look for | Action |
|---|---|---|---|
| Indexed pages | Indexing > Pages | Important pages excluded | Inspect URL and request indexing after fixes |
| Sitemap status | Sitemaps | `sitemap.xml` fetched successfully | Resubmit only after important route changes |
| Impressions | Performance > Search results | Queries gaining visibility | Add internal links or improve matching content |
| Clicks | Performance > Search results | Pages receiving real traffic | Prioritize CRO and speed checks |
| CTR | Performance > Search results | High impressions, low CTR | Rewrite title/meta only when intent mismatch is clear |
| Average position | Performance > Search results | Positions 8-20 | Strengthen content and internal links |
| Queries | Performance > Queries | New long-tail opportunities | Feed editorial roadmap |
| Countries | Performance > Countries | US/UK/Canada/Australia/EU mix | Prioritize English content and examples |
| Devices | Performance > Devices | Mobile CTR or position weakness | Check mobile UX and snippets |

## Query Classification

Classify new queries into:

- Tool intent: user wants to perform an action now.
- Guide intent: user wants instructions.
- Problem intent: user has an error or quality issue.
- Comparison intent: user compares tools or formats.
- Trust intent: user worries about upload, privacy or safety.

## CTR Rules

Do not rewrite titles after only a few impressions.

Rewrite only when:

- impressions are meaningful;
- the page ranks but CTR is weak;
- query intent differs from the visible title/description;
- the new copy remains honest.

## Indexing Rules

Request indexing manually only for:

- homepage;
- catalog pages;
- newly published high-priority guides;
- newly improved pages with material SEO changes.

Do not repeatedly submit unchanged pages.

## Monthly Review

Every 30 days:

1. Export top queries.
2. Export top pages.
3. Identify pages with impressions but no clicks.
4. Identify pages ranking 8-20.
5. Update the editorial roadmap with real query data.
6. Decide whether a new guide is justified.

## Guardrails

- Do not create content only because a keyword exists.
- Do not create pages that cannibalize tool pages.
- Do not imply OCR, Office conversion or cloud processing unless those features exist.
- Do not chase broad Adobe-level keywords before LiftPDF earns authority.
